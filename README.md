
On veut coder une fonction `number_to_ascii_art` telle que :

```python
number_to_ascii_art(1) == [
    '  #',
    '  #',
    '  #',
    '  #',
    '  #'
]

number_to_ascii_art(2) == [
    '###',
    '  #',
    '###',
    '#  ',
    '###'
]


number_to_ascii_art(10) == [
    '  #  ###',
    '  #  # #',
    '  #  # #',
    '  #  # #',
    '  #  ###'
]

number_to_ascii_art(235) == [
    '###  ###  ###',
    '  #    #  #  ',
    '###  ###  ###',
    '#      #    #',
    '###  ###  ###'
]
```
